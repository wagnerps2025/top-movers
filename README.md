# Top Movers — Varredura em tempo real

Painel web que varre pares (Binance) em tempo real, destacando movimentos acelerados de subida/descida e exibindo alertas recentes.

## Como usar
1. Abra `index.html` em um navegador moderno.
2. Ajuste a janela curta e limiares nos controles do topo.
3. Acompanhe os painéis Top 5 e os alertas.

## Configurações
- Janela curta: 1–3 minutos (ajustável)
- Limiar subida/descida (%)
- Máx. esticamento (%)
- Mín. liquidez (USD 24h)

## Dados
- Fonte: Streams públicos da Binance (`!miniTicker@arr`) via WebSocket.
- Este projeto é informativo e não constitui recomendação financeira.
- Não executa ordens.
